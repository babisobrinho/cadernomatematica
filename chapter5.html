introducao()

Uma matriz é um arranjo retangular de números, símbolos ou expressões, organizados em linhas e colunas. As matrizes são amplamente utilizadas em matemática, física, engenharia e ciência da computação.

Definicao()

Uma matriz A de dimensão m × n (lê-se "m por n") é um conjunto de números dispostos em m linhas e n colunas:

A = [a₁₁ a₁₂ ... a₁ₙ
     a₂₁ a₂₂ ... a₂ₙ
     ... ... ... ...
     aₘ₁ aₘ₂ ... aₘₙ]

O elemento aᵢⱼ está localizado na linha i e coluna j da matriz.

Exemplo()

Matriz 2×3:

A = [1  2  3
     4 -1  0]

Onde a₂₂ = -1 e a₁₃ = 3.

tiposMatrizes()

Existem vários tipos especiais de matrizes, cada um com propriedades particulares:

MatrizQuadrada

Número de linhas = número de colunas (n×n)

Exemplo: [1 2; 3 4]

MatrizIdentidade

Diagonal principal = 1, outros = 0

Exemplo: [1 0; 0 1]

MatrizNula

Todos elementos = 0

Exemplo: [0 0; 0 0]

MatrizDiagonal

Elementos fora da diagonal = 0

Exemplo: [2 0; 0 3]

TabelaComparacao()

Tipo Definição Exemplo

operacoes()

As operações com matrizes seguem regras específicas que diferem das operações com números reais.

AdicaoSubtracao()

Só possível entre matrizes de mesma dimensão:

A + B = [a₁₁+b₁₁ a₁₂+b₁₂ ... a₁ₙ+b₁ₙ
         a₂₁+b₂₁ a₂₂+b₂₂ ... a₂ₙ+b₂ₙ
         ... ... ... ...
         aₘ₁+bₘ₁ aₘ₂+bₘ₂ ... aₘₙ+bₘₙ]

Exemplo: [1 2] + [3 4] = [4 6]

MultiplicacaoEscalar()

Cada elemento é multiplicado pelo escalar:

k × A = [k×a₁₁ k×a₁₂ ... k×a₁ₙ
         k×a₂₁ k×a₂₂ ... k×a₂ₙ
         ... ... ... ...
         k×aₘ₁ k×aₘ₂ ... k×aₘₙ]

MultiplicacaoMatrizes()

Só possível se o número de colunas da primeira matriz = número de linhas da segunda:

Se A é m×n e B é n×p, então C = A×B é m×p
onde cᵢⱼ = Σ (aᵢₖ × bₖⱼ) para k de 1 a n

Exemplo: [1 2; 3 4] × [5 6; 7 8] = [19 22; 43 50]

Transposicao()

Troca linhas por colunas:

Aᵀ = [a₁₁ a₂₁ ... aₘ₁
      a₁₂ a₂₂ ... aₘ₂
      ... ... ... ...
      a₁ₙ a₂ₙ ... aₘₙ]

aplicacoes()

As matrizes têm diversas aplicações em várias áreas do conhecimento:

ComputacaoGrafica

  • Transformações geométricas (rotação, translação, escala)
  • Processamento de imagens (filtros, convoluções)

Fisica

  • Mecânica quântica (operadores, estados quânticos)
  • Relatividade (transformações de Lorentz)

Economia

  • Modelos input-output (análise de interdependência)
  • Otimização de portfólios

MachineLearning

  • Redes neurais (pesos e ativações)
  • Processamento de dados em lote
