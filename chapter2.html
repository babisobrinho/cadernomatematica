introducao()

A teoria de conjuntos é um ramo fundamental da matemática que estuda coleções de objetos chamados conjuntos. É a base para várias áreas da matemática e tem aplicações em ciência da computação, lógica e outras disciplinas.

ConceitoBasico()

Um conjunto é uma coleção bem definida de objetos distintos, chamados elementos.

  • Os conjuntos são geralmente denotados por letras maiúsculas (A, B, C...)
  • Os elementos são listados entre chaves { }
  • Exemplo: A = {1, 2, 3, 4}

FormasDefinicao()

Os conjuntos podem ser definidos de várias maneiras:

  • Extensão: Listando todos os elementos (A = {1, 2, 3})
  • Compreensão: Por uma propriedade (B = {x | x é par})
  • Diagramas de Venn: Representação gráfica

ConjuntosEspeciais()

Conjunto Notação Descrição
Vazio ∅ ou {} Não contém elementos
Unitário {a} Contém exatamente um elemento
Universal U Contém todos os elementos em consideração
Naturais ℕ {1, 2, 3, ...}
Inteiros ℤ {..., -2, -1, 0, 1, 2, ...}

definicoes()

Conceitos fundamentais da teoria de conjuntos que estabelecem as bases para operações e propriedades.

Pertenca()

Relação entre um elemento e um conjunto:

Pertence:

x ∈ A (x é elemento de A)

Exemplo: 2 ∈ {1, 2, 3}

Não pertence:

x ∉ A (x não é elemento de A)

Exemplo: 4 ∉ {1, 2, 3}

Subconjuntos()

Relação entre dois conjuntos:

Subconjunto:

A ⊆ B (todo elemento de A está em B)

Exemplo: {1, 2} ⊆ {1, 2, 3}

Subconjunto próprio:

A ⊂ B (A ⊆ B e A ≠ B)

Exemplo: {1, 2} ⊂ {1, 2, 3}

Igualdade()

Dois conjuntos são iguais se contêm exatamente os mesmos elementos:

A = B ⇔ (A ⊆ B ∧ B ⊆ A)

Exemplo:
{1, 2, 3} = {3, 2, 1} = {1, 1, 2, 3} (a ordem e repetições não importam)

operacoes()

Operações fundamentais que podem ser realizadas entre conjuntos para produzir novos conjuntos.

Interseção()

A ∩ B = {x | x ∈ A ∧ x ∈ B}

Elementos comuns a A e B

União()

A ∪ B = {x | x ∈ A ∨ x ∈ B}

Elementos de A ou de B

Diferença()

A \ B = {x | x ∈ A ∧ x ∉ B}

Elementos de A que não estão em B

Complementar()

Ac = U \ A

Elementos não pertencentes a A

DiferençaSimétrica()

A Δ B = (A \ B) ∪ (B \ A)

Elementos em A ou B, mas não em ambos

ProdutoCartesiano()

A × B = {(a,b) | a ∈ A ∧ b ∈ B}

Pares ordenados de elementos

Exemplos()

// Dados:
A = {1, 2, 3, 4}
B = {3, 4, 5, 6}
U = {1, 2, 3, 4, 5, 6, 7, 8}

// Operações:
A ∩ B = {3, 4}
A ∪ B = {1, 2, 3, 4, 5, 6}
A \ B = {1, 2}
B \ A = {5, 6}
A Δ B = {1, 2, 5, 6}
Ac = {5, 6, 7, 8}

Diagrama de Venn:

A
B
A ∩ B

propriedades()

Principais propriedades e leis que governam as operações com conjuntos.

LeisDeMorgan()

(A ∪ B)c = Ac ∩ Bc

A negação de "A ou B" é 
"não A e não B"
(A ∩ B)c = Ac ∪ Bc

A negação de "A e B" é 
"não A ou não B"

OutrasPropriedades()

Propriedade União Interseção
Comutativa A ∪ B = B ∪ A A ∩ B = B ∩ A
Associativa (A ∪ B) ∪ C = A ∪ (B ∪ C) (A ∩ B) ∩ C = A ∩ (B ∩ C)
Distributiva A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C) A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)
Idempotência A ∪ A = A A ∩ A = A
Elemento neutro A ∪ ∅ = A A ∩ U = A
Elemento absorvente A ∪ U = U A ∩ ∅ = ∅

aplicacoes()

A teoria de conjuntos tem aplicações em diversas áreas da matemática e ciência da computação.

BaseDeDados()

  • Operações de conjuntos correspondem a operações SQL (UNION, INTERSECT, EXCEPT)
  • Álgebra relacional é baseada na teoria de conjuntos

Programacao()

  • Estruturas de dados como conjuntos (Set) em várias linguagens
  • Operações booleanas em condições lógicas

Matematica()

  • Base para definições em probabilidade, relações, funções
  • Teoria dos números, álgebra, topologia usam conceitos de conjuntos

Logica()

  • Correspondência entre operações lógicas e de conjuntos
  • Demonstrações matemáticas usando teoria de conjuntos
